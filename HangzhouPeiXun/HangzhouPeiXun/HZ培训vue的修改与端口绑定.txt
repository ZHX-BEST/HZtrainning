在build所在文件夹下打开cmd:
npm run dev //进入开发模式，前端重新分配端口，可以修改
node build/build.js //编译模式，前后端绑定，不可修改
**进入编译模式后将生成的dist文件夹下的static和index.html更新到项目文件夹下

^^^^^^npm run dev ERR!错误时，重新安装webpack^^^^^^^^
（1）依次输入以下命令
npm uninstall webpack-dev-server
（2）安装2.9.1的webpack
npm install webpack-dev-server@2.9.1
（3）运行项目
npm run dev
